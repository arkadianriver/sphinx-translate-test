Sphinx翻訳テスト¶

こんにちは世界！ここにいくつかのドキュメントがあります。

`QuickStart Guide`_を使用し、翻訳po_を使用してSphinx-intlをテストします。

`github pages`_で翻訳された出力を参照してください。

pip install sphinx sphinx-intl
pip install git+https://github.com/arkadianriver/translate-po.git@recursive-src-dest#egg=translate-po

「run.sh`」または「run.bat`」コマンドを実行して、テキストをPOファイルに抽出し、それらを翻訳し、ローカライズされたバージョンに構築します。
